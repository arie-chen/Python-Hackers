Installing Relevant Software:

In terminal:
"pip" (or "pip3" for macOS) is used to retrieve python libraries. Ensure that Python3 and pip3 are installed on your operating system. The libraries required are pypdf2, tabula-py[cv], camelot and requests. Open a terminal window and run the following commands.

pip3 requests 
pip3 pypdf2 
pip3 tabula-py[cv]
pip3 camelot
